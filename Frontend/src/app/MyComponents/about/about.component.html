About Task Manager

Welcome to Task Manager – your smart and simple solution to stay organized and productive.

This application helps users easily create, manage, and track their daily tasks. Whether you’re an individual managing your personal to-dos or a team leader assigning tasks, Task Manager is here to simplify your workflow.

Key Features:
  • Create and edit tasks
  • Filter tasks based on their status (Pending, Completed)
  • User login and registration
  • Admin panel for managing users
  • Update your profile information

Built with ❤️ using Angular for frontend and .NET for backend.

Thank you for using Task Manager! We hope it helps you get things done more efficiently.
